# Day 80 - GitHub User Finder

## Overview
The Github user finder is a web application that allows users to search for Github profiles by username and visualize key information about the user, such as their avatar, number of repositories, followers, and following count and more.

## Features
- Search for Github users by username.
- Display user profile information including avatar, name, bio, location, and more.
- Show statistics such as the number of repositories, followers, and following.
- Responsive design for optimal viewing on various devices.
- Error handling for non-existent users.

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API for making requests to the Github API
