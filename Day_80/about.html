About GitHub User Finder

What is GitHub User Finder?

GitHub User Finder is a simple yet powerful web application that lets you search for GitHub users and instantly view their complete profile information. Built as part of the 100 Days JavaScript Challenge, this project demonstrates API integration and modern web development practices.

Features

  • User Search - Search for any GitHub user by username
  • Profile Visualization - View user avatar, bio, and account statistics
  • Statistics - See followers, following count, and public repositories
  • Top Languages - Discover the programming languages used most frequently
  • Recent Repositories - Browse up to 6 recently updated repositories
  • Repository Details - View repository descriptions, language, stars, and forks
  • Direct Links - Quick access to GitHub profiles and repositories

Technologies Used

  • HTML5 - Semantic markup and structure
  • CSS3 - Professional styling with dark theme
  • JavaScript (ES6+) - Async/await, modules, and DOM manipulation
  • GitHub API v3 - User and repository data endpoints

How It Works

  1. Enter a GitHub username in the search bar
  2. Click Search or press Enter to fetch the user's data
  3. View the user's profile card with complete information
  4. Explore their top programming languages and recent repositories
  5. Click "View Profile on GitHub" to visit their actual GitHub page

Learning Objectives

This project was created to master:

  • Asynchronous JavaScript (async/await, Promises)
  • REST API integration and data fetching
  • Module system (import/export)
  • DOM manipulation and dynamic content rendering
  • CSS Grid and Flexbox for responsive layouts
  • Error handling and user feedback
