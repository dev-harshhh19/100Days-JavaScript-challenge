# Day 64 Task

## Objective

Read and parse student data from a local JSON file using Node.js file system operations.

## Requirements

- Read `data.json` using the fs/promises module
- Parse the JSON content
- Display the department name
- List all student names

## Implementation

The task demonstrates async file operations with proper error handling using try-catch blocks.

## Running the Task

```bash
node script.js
```

## Expected Output

```
Department: [department name]
[student 1 name]
[student 2 name]
...
```
