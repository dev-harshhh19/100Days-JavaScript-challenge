About Weather App

Overview

A modern, real-time weather application built with clean, modular JavaScript. Get accurate weather data with a beautiful, responsive interface.

Key Features

  • Real-time weather from OpenWeatherMap API
  • 7-day forecast
  • Temperature unit conversion
  • Dark/Light theme
  • Recent search history
  • Weather animations
