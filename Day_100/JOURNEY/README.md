# 100 Days of JavaScript - Visual Journey

An interactive visual celebration of completing the 100 Days JavaScript Challenge.

## Features

- **Animated Hero** - Counting stats, particles background
- **Timeline** - 4 learning phases with milestones  
- **Projects Grid** - SVG icons for each project built
- **Confetti Celebration** - Triggers at the end
- **Smooth Page Transitions** - Fade animations between pages

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Main visual journey with timeline |
| `roadmap.html` | Detailed 100-day roadmap |

## Run Locally

```bash
cd Day_100/JOURNEY
npx serve
```

Open `http://localhost:3000`
