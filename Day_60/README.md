# Day 60 - Dynamic Calculator

A fully functional calculator web application with basic and advanced operations, calculation history, and theme support.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, and division
- **Advanced Operations**: Square root and percentage calculations
- **Clear Function**: Reset the display to start a new calculation
- **Calculation History**: View all previous calculations with persistent localStorage storage
- **Light/Dark Theme**: Toggle between light and dark modes with smooth transitions
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Error Handling**: Catches and displays calculation errors gracefully

## Files

- `calculator.html` - HTML structure with calculator grid layout
- `calculator.css` - Styling with CSS variables for theme support and responsive design
- `calculator.js` - Core calculator logic with history management and localStorage

## How to Use

1. Open `calculator.html` in your browser
2. Enter numbers using the number buttons
3. Select an operation (+, -, *, /)
4. Press equals (=) to see the result
