# Day 51 - Todo List in JavaScript

## Objective
- Create a simple Todo List application using JavaScript, HTML, and CSS.
- Learn how to manipulate the DOM and handle user input.

## Instructions
- Set up a basic HTML structure with an input field, a button to add tasks, and a list to display the tasks.
- Use JavaScript to add functionality:
  - When the user enters a task and clicks the "Add" button, the task should be added to the list.
  - Each task in the list should have a "Delete" button to remove the task from the list.
  - Optionally, add a feature to mark tasks as completed.
- Style the application using CSS to make it visually appealing.
